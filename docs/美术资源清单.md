# 美术资源清单（占位图 → 待统一生成替换）

> 当前游戏所有图标用 emoji / 色块占位。本清单列出**需要的美术资源**：资源名、用途、参考样式。
> 后续统一生成图片，按"资源名"放到 `packages/client/public/art/` 并替换 `client/src/info.ts` 里的 emoji 字段。
>
> **统一规范**：
> - 格式 PNG（透明背景），图标类 128×128，场景类按需。
> - 命名：`类别_名称.png`，全小写下划线，如 `res_wood.png`、`unit_legionnaire.png`。
> - 风格基调：罗马/古典战争策略，暖色木质 UI，写实偏插画。可后续在此统一指定色板。

## 替换流程（生成图后）

1. 把图片放进 `packages/client/public/art/`（按上面文件名）。
2. 改 `client/src/info.ts`：把 `icon: '🪵'` 改为 `icon: '/art/res_wood.png'`，并在渲染处把 emoji 文本改为 `<img>` 标签（届时我统一改一次渲染函数，支持 emoji 与图片两种）。
3. 无需改任何后端代码——美术是纯前端展示层。
