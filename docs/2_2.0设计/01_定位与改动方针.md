## 一、原版怎么做

Travian 的内核（拆解结论）：

- **品类**：服务器权威、实时、持续运行、赛季制（约 200 天）的多人 PvP 策略战争游戏。
- **核心循环**：产资源 → 建建筑 → 爆兵 → 扩张/掠夺 → 联盟协作 → 终局争夺世界奇迹。
- **设计哲学**：时间即资源 / 资源是硬约束 / 不可单人通关 / 掠夺驱动 PvP / 赛季重置。
- **代价**：上手门槛高、极度肝、时区焦虑、单人不友好、赛季末断崖流失。

## 二、2.0 需要先回答的 4 个定位问题

这一节不谈具体系统，只定**方向坐标**。后面每个系统都会引用这里的答案。

### Q1. 目标玩家是谁？（决定肝度与社交强度）
- **A. 硬核 SLG 老玩家**：要深度、要 PvP、能接受肝。靠拢原版。
- **B. 休闲/泛策略玩家**：要轻松、碎片化、单人也能玩爽。大改原版。
- **C. 两者兼顾（分服或分阶段）**：休闲服 + 硬核服，或前期休闲后期硬核。

### Q2. PvP 的强制程度？（决定"掠夺驱动"是否保留）
- **A. 强 PvP**：保留掠夺、占领、抢神器，弱肉强食（原版）。
- **B. 弱 PvP / 可选 PvP**：PvP 限定区域或时段，新手/休闲玩家可避战，PvE 也能成长。
- **C. PvE 为主，PvP 为点缀**：偏离 Travian 较远。

### Q3. 社交的强制程度？（决定"不可单人通关"是否保留）
- **A. 强制联盟**：终局必须靠联盟，单人无法获胜（原版）。
- **B. 联盟增益但非必须**：单人有独立成长线/单人成就/PvE 终局，联盟是"更强但非唯一"。
- **C. 单人优先**：联盟仅作社交，不影响主线。

### Q4. 节奏与终局形态？（决定赛季长度与重置）
- **A. 长赛季 + 世界奇迹 + 全清重置**（原版）。
- **B. 短赛季（数周）+ 多元胜利 + 部分继承**（降低流失）。
- **C. 无限世界 / 不重置**（彻底改 Travian 模型，留存导向）。

### 关键设计决策：PvE / PvP 同源

这是 2.0 区别于原版理解方式的核心定调：

- **一支军队、一套数值**：玩家养成的军队既能打 PvE 目标，也能打 PvP 目标，用同一套兵种属性、战斗公式、损耗规则。不存在"PvE 专用兵"和"PvP 专用兵"。
- **两条发育速率**：
  - **PvE = 发育地板（稳定积累）**：打 NPC 目标（野怪/绿洲/NPC 营地/冒险）能稳定、低风险地获得资源与成长，保证不掠夺也能玩下去。
  - **PvP = 发育加速（高风险高回报）**：掠夺其他玩家获得更快但更不稳定的资源，承担被反打、损兵的风险。
- **含义**：PvE 是"保底线"，PvP 是"加速器"。玩家可以纯 PvE 慢慢发育，也可以靠 PvP 抢跑——但两者都喂给同一个经济/军队循环。

### 据此收敛的核心循环（取代上面的草案）

```
        ┌────────────────── 资源 ──────────────────┐
        │  (资源田产出 + 战利品回流)                  │
        ▼                                            │
   建造/升级建筑 ──→ 训练军队 ──→ 一支军队两种用途      │
   (发育主轴/科技)        │            │              │
                         │     ┌──────┴───────┐      │
                         │     ▼              ▼      │
                         │  打 PvE 目标     打 PvP 目标 │
                         │ (稳定积累)      (掠夺加速)   │
                         │     │              │      │
                         │     └──────┬───────┘      │
                         │            ▼              │
                         │      战利品 + 经验 ─────────┘
                         │            │
                         └────────────┘  军队/英雄变强 → 能打更高目标
```

一句话：**经济造军队，军队打目标(PvE稳/PvP快)，战利品回流经济，循环放大**。建筑科技是发育主轴，英雄是成长杠杆。
