# 兵种特性效果表（TraitEffect 枚举参考）

> 新增特性时，在 `config/unit_traits.csv` 中填入 effect 代码即可。  
> **新增 effect 类型**（代码里没实现过的）需要同时改代码：
> 1. `packages/server/src/infra/combat-types.ts` — 在 `TraitEffect` union 里添加新 key
> 2. 在对应模块（`combat.ts` 或 `military.ts`）里实现公式
> 3. 回到本文档更新这张表

---

## 已实现的 effect key

| effect key | 作用位置 | 公式 | 说明 |
|---|---|---|---|
| `atk_melee` | `combat.ts` | `damage × (1 + value)` | 近战攻击力加成（正值=加，负值=减） |
| `atk_ranged` | `combat.ts` | `damage × (1 + value)` | 远程攻击力加成 |
| `def_melee` | `combat.ts` | `effectiveHP × (1 + value)` | 近战防御加成，等效于增加血量 |
| `def_ranged` | `combat.ts` | `effectiveHP × (1 + value)` | 远程防御加成 |
| `dmg_taken_melee` | `combat.ts` | `effectiveHP ÷ (1 + value)` | 承受近战伤害修正。负值=减伤（如 `-0.20` → 少受 20% 近战伤害） |
| `dmg_taken_ranged` | `combat.ts` | `effectiveHP ÷ (1 + value)` | 承受远程伤害修正。负值=减伤（如 `-0.30` → 持盾兵少受 30% 远程伤害） |

---

## CSV 格式

`config/unit_traits.csv` 支持最多 **5 组** effect/value（effect1/value1 … effect5/value5），空列自动忽略。

```csv
id,code,name,effect1,value1,effect2,value2,effect3,value3
1,shield,持盾,dmg_taken_ranged,-0.30,,,,
2,heavy_armor,重甲,def_melee,0.10,dmg_taken_ranged,-0.15,,
```

`units.csv` 的 `traits` 列用数字 ID（逗号分隔）引用特性，例如 `traits=1,2`。

## 添加新 effect 的完整步骤

1. `combat-types.ts` → `TraitEffect` union 加新 key，同时在 `TRAIT_EFFECTS` 数组加同名字符串
2. 选择实现位置：
   - 战斗属性（攻/防/伤害）→ `modules/combat.ts` 里 `traitMult()` 调用处
   - 行军属性（速度/携带量）→ `modules/military.ts` 里 `finalStats()`，把乘法叠到对应字段上
3. 在本文档的表格里补一行
