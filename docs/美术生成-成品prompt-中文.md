# 美术生成 · 成品 Prompt（中文 · 可直接投喂）

> 用法：一批 = 图像模型里开一个新会话，整段贴进去出图。建议先出③建筑或④兵种定调，
> 挑一张满意的设为「风格参考图」，再带出其余各批，一致性最好。
>
> **所有图统一保存到**：`packages/client/public/art/`，文件名与每批下方标注一致（`.png`），
> 覆盖同名占位图即可，无需改任何代码。跑 `npm run build` 生效。
>
> 视觉基调：通用中世纪 / 奇幻。文件名/中文名以 `packages/client/src/info.ts` 为准。
> 规格详见 [美术生成规范.md](./美术生成规范.md)。
