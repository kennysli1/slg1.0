# 03 · 军事与战斗系统

> 还原 Travian 的战斗骨架：兵种 → 训练/科技 → 战斗公式 → 攻城 → 英雄 → 掠夺。

## 1. 兵种（Troops）

- 全服约 **50 种兵**，每部族约 **10 种**。
- 每个兵种有属性：**攻击力 / 对步防 / 对骑防 / 速度 / 载货量 / crop消耗 / 造价**。
- 分类（按职能）：

| 职能 | 说明 | 代表 |
|------|------|------|
| 步兵 Infantry | 兵营训练，便宜量大 | 罗马军团兵、条顿棍棒兵 |
| 骑兵 Cavalry | 马厩训练，速度快、载货高、贵 | 罗马禁卫骑兵、匈奴骑兵 |
| 侦察 Scout | 侦查敌村资源/兵力/防御 | 各族斥候 |
| 攻城器械 Siege | 兵工厂训练，破坏建筑/城墙 | 投石机(打建筑)、攻城锤(打墙) |
| 行政官 Administrator | 降低敌村忠诚度以占领 | 元老/酋长/首领 |
| 拓荒者 Settler | 建立新村（3个） | 各族拓荒者 |

> 兵种差异 + 部族差异是战斗深度与 build 多样性的来源。

## 2. 训练与科技

- **兵营/马厩/兵工厂**：等级越高训练越快，可排队。
- **学院 Academy**：研究解锁高级兵种。
- **铁匠铺 Smithy**：逐级提升某兵种的攻/防数值（投入资源换永久强化）。
- 训练受 crop 净产出约束（养不起就不能无限爆兵）。

## 3. 战斗系统（核心公式骨架）

Travian 战斗为**一次性结算**（行军到达即结算，非回合制对拼）。简化模型：

```
进攻总攻击力 = Σ(各兵种数量 × 攻击力) × 英雄/装备/士气加成
防守总防御力 = Σ(各兵种数量 × 对应防御) × 城墙加成 × 防御建筑加成

胜负与损失比例 由 攻防比 决定（非线性，强者损失小、弱者全灭）
```

关键机制：
- **攻击类型**：普通攻击(Normal) vs 突袭/掠夺(Raid)。Raid 损失更小、只抢资源不占村，是刷资源主玩法。
- **城墙加成**：防守方城墙等级提供百分比防御加成（罗马城墙加成最高）。
- **士气与人口加成 Morale**：大号打小号有惩罚，保护新手。
- **攻城阶段**：先打城墙/再打建筑；投石机可指定摧毁特定建筑。
- **侦察战**：斥候 vs 斥候，决定能否拿到情报、是否暴露。
- **再侦/连击**：行军波次（wave）与到达时间的精确控制是高玩核心技巧（"清防+占领"连击）。

## 4. 英雄系统（Hero）

- 每账号 1 个英雄，是特殊强力单位，可随军提供攻/防加成。
- **属性点**：战斗力 / 攻防加成 / 资源产出 / 经验加成，可自由分配。
- **冒险 Adventures**：地图上刷新冒险点，英雄出征获得经验与**装备/道具**。
- **装备**：武器、护甲、坐骑、消耗品（加速、扩载、复活等），可在**银币拍卖市场**交易。
- **绿洲清野**：英雄带兵清绿洲野怪后占领绿洲（接 `02` 经济）。
- 英雄死亡可在英雄殿(Hero's Mansion / 复活)重生。

## 5. 掠夺与防守经济（PvP 引擎）

- **掠夺 Farming**：用 Raid 反复抢弱村资源，是中期资源主来源。高玩维护"农场列表(Farm List)"批量发兵。
- **防守手段**：城墙、驻防军、隐藏地窖(Cranny)、联盟驰援、避战（把资源花光/藏起）。
- **驰援 Reinforce**：把兵派到友军村庄共同防守，是联盟协防核心。

## 平衡基线（占位，需校准）

- 兵种三维（攻/步防/骑防）遵循"克制三角"思路：步兵、骑兵、攻城各有强弱。
- 速度单位 = 格/小时；行军时间 = 距离 / 速度（受英雄坐骑、神器影响）。
- 掠夺保护：地窖容量、士气惩罚、新手保护期（注册后若干天免战）。
