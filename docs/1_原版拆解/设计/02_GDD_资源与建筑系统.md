# 02 · 资源与建筑系统（经济）

> 还原 Travian 的经济骨架：资源 → 资源田 → 建筑 → 存储 → 交易 → 扩张。

## 1. 资源（Resources）

四种基础资源 + 人口约束：

| 资源 | 产自 | 主要用途 | 部族偏好 |
|------|------|----------|----------|
| 木 Wood | 伐木场 Woodcutter | 建筑、兵种 | 条顿耗木 |
| 泥 Clay | 泥坑 Clay Pit | 建筑、兵种 | 高卢耗泥 |
| 铁 Iron | 铁矿 Iron Mine | 兵种、武器 | 罗马耗铁 |
| 粮 Crop | 农田 Cropland | **养人口/军队（净产出）** | 全局约束 |

- **Crop 是净值**：每个建筑、村民、士兵都消耗 crop。若 crop 净产出为负，仓库被吃空后军队会"饿死"。这是 Travian 最硬的经济约束。
- 资源按**每小时产量**实时累积，存满即溢出（浪费）。

## 2. 资源田（Resource Fields）

- 每座村庄外围有 **18 块资源田**（标准村：4木/4泥/4铁/6粮，分布因地图位置而异）。
- 每块可独立升级（通常 1–10 级，部分玩法到 20 级），等级越高产量越高、耗费越高。
- **绿洲 Oasis**：地图上的特殊地块，占领后给村庄提供资源产量加成（如 +25%/+50% 某资源）。需要英雄清野并建"宫殿/官邸"占领。

## 3. 建筑（Buildings）

每座村庄中心有约 **22 个建筑槽位**。建筑分三类：

### 3.1 基础设施类
| 建筑 | 功能 |
|------|------|
| 主基地 Main Building | 降低所有建筑建造时间（等级越高越快） |
| 仓库 Warehouse | 提升木/泥/铁存储上限 |
| 粮仓 Granary | 提升 crop 存储上限 |
| 市场 Marketplace | 资源交易、向他村运送资源 |
| 大使馆 Embassy | 加入/创建联盟，提升联盟容量 |
| 集结点 Rally Point | 派兵、查看进出军队 |
| 隐藏地窖 Cranny | 隐藏部分资源不被掠夺（条顿地窖效果减半 = 反制） |

### 3.2 军事类（详见 `03`）
| 建筑 | 功能 |
|------|------|
| 兵营 Barracks | 训练步兵 |
| 马厩 Stable | 训练骑兵 |
| 兵工厂 Workshop | 训练攻城器械 |
| 学院 Academy | 研究新兵种 |
| 铁匠铺 Smithy | 升级兵种攻防 |
| 城墙 City Wall / 土墙 / 栅栏 | 防御加成（部族不同外观与系数） |
| 王宫 Palace / 官邸 Residence | 防内奸、产文化点、扩张/占领村庄 |

### 3.3 进阶/经济类
| 建筑 | 功能 |
|------|------|
| 谷物磨坊 / 面包房 | 提升 crop 产量百分比 |
| 锯木厂 / 砖窑 / 铸铁厂 | 提升对应资源产量百分比 |
| 贸易办公室 | 提升商队运载量 |
| 宝物库 Treasury | 存放神器（终局） |
| 世界奇迹 Wonder of the World | 终局胜利建筑（仅特殊村） |

- **前置依赖**：许多建筑需要其他建筑达到特定等级才能建造（科技树）。
- **建造队列**：原版免费仅 1 条队列（Gold/Plus 可加队列），队列管理是核心体验摩擦点。

## 4. 人口与文化点（扩张）

- **人口 Population**：建筑/资源田等级提升人口，人口决定排名与 crop 消耗。
- **文化点 Culture Points (CP)**：由建筑产出，**累计到阈值才能建立/占领新村**。
  - 第 1 个新村需 X CP，第 2 个需更多，依次递增 → 控制扩张节奏。
- **扩张方式**：
  1. **建立新村 Settle**：训练 3 个"拓荒者(Settler)"+ 找空地建村。
  2. **占领他村 Conquer**：用"元老/酋长/管理者(Administrator)"降低目标村忠诚度到 0 夺取。

## 5. 市场与交易

- 玩家间运送资源（受商队运载量与距离/时间限制）。
- 资源兑换（受比例限制，防套利）。
- 联盟内援助、英雄装备的"银币市场"（拍卖）。

## 平衡基线数值（可调，正式开发前需校准）

> 以下为经验还原值，作为公式占位，**非精确原版数值**。

- 资源田产量（每小时，约略）：Lv1≈基础值，每级约 ×1.3~1.5 递增。
- 建造耗费随等级指数增长（约 ×1.28/级），建造时间同理（受主基地等级削减）。
- 仓库/粮仓容量随等级阶梯增长。
- CP 扩张阈值随村庄数递增（第 n 村阈值远大于第 n-1 村）。
